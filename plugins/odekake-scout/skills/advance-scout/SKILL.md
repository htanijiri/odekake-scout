# advance-scout Skill — 先行予約調査

## 役割

1〜2ヶ月先に開催される子ども向けイベントのうち、事前予約や抽選申込が必要なものを調査し、予約情報をまとめて出力する。

## 家族プロフィール

以下のコマンドを実行してprofile.ymlを読み込むこと。これは必須ステップである。

```
cat profile.yml
```

このコマンドでファイルの内容が表示されたら、その内容を家族プロフィールとして使用する。

ファイルが見つからない場合は、以下のメッセージを表示して処理を停止する:
「profile.yml が見つかりません。カレントディレクトリに profile.yml を作成してください。書き方の例は README.md の「カスタマイズ」セクションを参照してください。」

デフォルト値は使用しない。必ず profile.yml から読み込むこと。

## 検索戦略

1. 「{エリア} 子ども イベント {来月の月名}」で各エリアを検索
2. 「{興味} イベント {来月の月名} 関東 チケット 予約」で興味別に検索
3. 検索結果から予約・抽選が必要なイベントをピックアップ
4. 各イベントの予約開始日・申込方法・締切を調査
5. 結果が少なければクエリを変えてリトライ

## 出力形式

- 申込方法: {Web / 電話 / 抽選}

### {イベント名}

（予約不要の大型イベントも参考としてリスト化）

## ルール

- 実在するイベントのみ掲載する。架空のイベントや日程を作らない
- 予約開始日・締切が不明な場合は「要確認」と明記する
- 抽選制のイベントは当選確率の目安があれば記載する
- 公式ページのURLを必ず含める
- 検索結果をそのまま信頼せず、日付や料金は公式情報と照合する
